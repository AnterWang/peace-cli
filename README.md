### 文件资源目录

```text
├─ bin
│  ├─ cli                             # 命令和参数
├─ package.json                       # 依赖包管理
│  ├─ dependencies       
│  ├─ ├─ commander                    # 自定义命令行指令   
│  ├─ ├─ fs-extra                     # fs-extra 是对fs模块的扩展，支持promise    
│  ├─ ├─ chalk                        # 命令行美化工具 
│  ├─ ├─ figlet                       # 命令行美化工具（绘制图像）
│  ├─ ├─ inquirer                     # 交互
│  ├─ ├─ ora                          # 命令行 loading
│  ├─ ├─ download-git-repo            # 下载远程模版工具包
├─ .gitignore                         # 代码上传忽略文件
├─ README.md                          # README 介绍
├─ yarn.lock                          # 依赖包包版本锁
```
